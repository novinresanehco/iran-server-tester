# 🇮🇷 Iran Server Tester v3

ابزار پیشرفته تحلیل کیفیت سرور برای اینترنت ایران
مخصوص انتخاب هوشمند پروتکل و پنل (Reality / DNS Tunnel / Xray)

---

## 🎯 هدف پروژه

در شرایط فعلی اینترنت ایران (۲۰۲۶)، بسیاری از روش‌های سنتی مانند Reality یا WS دیگر پایداری سابق را ندارند.

این اسکریپت با انجام تست‌های واقعی شبکه، به شما کمک می‌کند:

* ❌ از سرورهای نامناسب استفاده نکنید
* ✅ بهترین پروتکل واقعی (نه تئوری) را انتخاب کنید
* 🚀 قبل از نصب، نتیجه قطعی بگیرید

### 🧬 Phase 6 — DNS Tunnel (جدید در v3)

📢 مهم‌ترین تغییر نسخه 3

در آوریل 2026:

> تنها روش پایدار در ایران = DNS Tunnel

این فاز بررسی می‌کند:

---

#### ✅ موارد تست:

* باز بودن پورت 53
* پاسخ‌دهی DNS سرور
* پشتیبانی از query کوتاه
* بررسی رفتار DPI

---

#### ⚙️ نکته مهم (MasterDNS)

بر اساس توصیه توسعه‌دهنده:

```bash
max-qname-len=101
```

📌 این مقدار:

* باعث عبور بهتر از DPI می‌شود
* طول درخواست DNS را بهینه می‌کند

---

#### 🧪 نتیجه:

اسکریپت مشخص می‌کند:

* آیا DNS Tunnel جواب می‌دهد یا نه
* آیا سرور برای MasterDNS مناسب است یا نه
